# ProtectEyes

**说明**:定时弹窗,强制休息,保护眼睛,防猝死。
* 之所以叫"强制休息"，是因为正常途径关不掉
* 麻烦一点，其实也可以关掉。比如电脑重启，或者用任务管理器来关闭。

![效果图](./Image/效果图.png)

## 配置

VS2015,win7

## 使用

Bin\设置开机自启.bat,可以将程序复制到C盘,并设置开机自启

运行程序后,每过1小时,弹窗就会出现,开始倒计时.倒计时结束前,窗体无法关闭、拖动、拉伸
